### How to Play
Seriously !?

### Build
git clone this project, then.
```bash
cd cheerzorboom
npm install
npm start
```
Head to [localhost:3000](localhost:3000).

#### `Blocks`:
An immutable Map represent the whole game board blocks, with `BlockRecord` as keys and `Block` as values.
